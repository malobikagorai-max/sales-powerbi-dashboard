# 📊 Sales Dashboard using Power BI

## 🔹 Project Overview
This project presents an interactive sales dashboard built in Power BI using retail sales data. The dashboard highlights overall sales performance, profit, total orders, product category performance, regional sales distribution, and time-based sales trends.

## 🔹 Dashboard Preview
![Dashboard](powerbi-dashboard.png)

## 🔹 Key Dashboard Features
- KPI cards for Total Sales, Total Profit, and Total Orders  
- Bar charts for category-wise and region-wise sales comparison  
- Line chart for sales trend over time  
- Interactive slicers for Product Category and Region  

## 🔹 Tools Used
- Power BI  
- Data Visualization  
- Basic Data Modeling  

## 🔹 Key Insights
- Total sales reached approximately **14.92M**  
- Total profit was approximately **1.52M**  
- Total order count was approximately **8.399K**  
- Technology generated the highest sales among product categories  
- Central region contributed the highest sales among regions  
- Sales showed fluctuations over time across months

Download the `.pbix` file and open it in Power BI Desktop to explore the interactive dashboard.
